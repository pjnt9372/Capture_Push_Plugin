# 🔍 `config.ini` 各部分详解

## 📌 `[logging]`

**作用**：配置应用程序的日志级别

| 配置项     | 值                     | 说明                                        |
|-----------|-----------------------|-------------------------------------------|
| `level`   | `DEBUG`  <br/> `INFO` <br/> `WARNING` <br/> `ERROR` <br/> `CRITICAL` | `DEBUG` 包含最详细的日志信息，主要用于调试<br/> `INFO` 一般信息，确认程序按预期工作<br/> `WARNING` 表示潜在的问题，但程序仍可继续运行<br/> `ERROR` 表示错误事件，程序某些功能无法执行<br/> `CRITICAL` 表示严重错误，程序可能无法继续运行 |

## 📌 `[run_model]`

**作用**：避免开发时重复拉取造成封禁

| 配置项     | 值                     | 说明                                        |
|-----------|-----------------------|-------------------------------------------|
| `model`   | `DEV`  <br/> `BUILD` | `DEV` 处于开发模式不反复拉取<br/> `BUILD` 构建模式正常拉取 |

## 📌 `[account]`

**作用**：存储用户账户信息（敏感信息已加密存储）

| 配置项     | 说明                                        |
|-----------|-------------------------------------------|
| `school_code` | 院校代码，用于确定使用哪个院校模块 |
| `username`    | 教务系统登录用户名 |
| `password`    | 教务系统登录密码（已加密） |

## 📌 `[push]`

**作用**：配置消息推送方式（用户有且只能同时启用一种推送方式）

| 配置项     | 值                     | 说明                                        |
|-----------|-----------------------|-------------------------------------------|
| `method`  | `none` <br/> `email` <br/> `feishu` <br/> `serverchan` | `none` 不启用任何推送<br/> `email` 邮件推送（需配置 `[email]` 节）<br/> `feishu` 飞书机器人推送（需配置 `[feishu]` 节）<br/> `serverchan` Server酱推送（需配置 `[serverchan]` 节） |

## 📌 `[email]`

**作用**：邮件推送的配置信息（当 `[push].method=email` 时生效）

| 配置项     | 说明                                        |
|-----------|-------------------------------------------|
| `smtp`    | SMTP 服务器地址（如 smtp.qq.com）          |
| `port`    | SMTP 端口（通常为 465 或 587）             |
| `sender`  | 发件人邮箱地址                              |
| `receiver`| 收件人邮箱地址                              |
| `auth`    | 邮箱授权码（非邮箱登录密码，已加密存储）    |

## 📌 `[feishu]`

**作用**：飞书机器人推送的配置信息（当 `[push].method=feishu` 时生效）

| 配置项     | 说明                                        |
|-----------|-------------------------------------------|
| `webhook_url` | 飞书机器人 Webhook 地址（已加密存储）     |
| `secret`      | 飞书机器人密钥，用于签名校验（已加密存储）  |

## 📌 `[serverchan]`

**作用**：Server酱推送的配置信息（当 `[push].method=serverchan` 时生效）

| 配置项     | 说明                                        |
|-----------|-------------------------------------------|
| `sendkey` | Server酱 SendKey（已加密存储）              |

## 📌 `[loop_getCourseGrades]`

**作用**：成绩循环检测配置

| 配置项     | 说明                                        |
|-----------|-------------------------------------------|
| `enabled` | 是否启用成绩循环检测（True/False）          |
| `time`    | 检测间隔时间（秒）                          |

## 📌 `[loop_getCourseSchedule]`

**作用**：课表循环检测配置

| 配置项     | 说明                                        |
|-----------|-------------------------------------------|
| `enabled` | 是否启用课表循环检测（True/False）          |
| `time`    | 检测间隔时间（秒）                          |

## 📌 `[schedule_push]`

**作用**：课表定时推送配置

| 配置项             | 说明                                        |
|------------------|-------------------------------------------|
| `today_8am`      | 是否在每天上午8点推送当天课表（True/False）  |
| `tomorrow_9pm`   | 是否在每天晚上9点推送明天课表（True/False）  |
| `next_week_sunday` | 是否在每周日推送下周课表（True/False）      |

## 📌 `[school_time]`

**作用**：学校作息时间配置

| 配置项             | 说明                                        |
|------------------|-------------------------------------------|
| `morning_count`  | 上午课程节数                                |
| `afternoon_count`| 下午课程节数                                |
| `evening_count`  | 晚上课程节数                                |
| `class_duration` | 每节课时长（分钟）                          |
| `first_class_start` | 第一节课开始时间（HH:mm格式）              |
| `class_times`    | 各节课具体时间，逗号分隔                      |

## 📌 `[semester]`

**作用**：学期相关信息配置

| 配置项             | 说明                                        |
|------------------|-------------------------------------------|
| `first_monday`   | 学期第一周周一的日期（YYYY-MM-DD格式）       |
